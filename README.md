# 인서울 챌린지

JavaFX로 제작한 간단한 육성 시뮬레이션 게임입니다.  
플레이어는 30일 동안 공부와 휴식을 반복하며 능력치를 올리고,  
최종적으로 목표 대학 합격 엔딩에 도전하게 됩니다.

## 프로젝트 소개

이 프로젝트는 JavaFX를 활용해 직접 만든 간단한 게임입니다.  
하루 동안 제한된 턴 안에서 과목 공부 또는 휴식을 선택할 수 있으며,  
체력과 능력치를 관리하면서 30일 동안 성장하는 구조로 만들었습니다.

게임이 끝나면 국어, 영어, 수학, 물리, 화학 능력치에 따라  
서울대, 연세대, 고려대 등 다양한 대학 합격 엔딩이 출력됩니다.

## 주요 기능

- JavaFX 기반 GUI 게임 화면 구현
- 메인 메뉴 화면 구성
- 배경 이미지 / 캐릭터 이미지 출력
- 배경 음악 재생 및 ON/OFF 기능
- 하루 최대 10턴 행동 가능
- 랜덤 과목/휴식 버튼 생성
- 공부 선택 시 능력치 증가, 체력 감소
- 휴식 선택 시 체력 회복
- 30일 진행 후 최종 엔딩 출력
- 능력치 조건에 따른 대학 합격 판정

## 게임 규칙

- 하루에 최대 10번 행동할 수 있습니다.
- 매 턴마다 과목 또는 휴식 중 일부가 랜덤으로 제시됩니다.
- 공부를 하면 해당 과목 능력치가 증가하지만 체력이 감소합니다.
- 휴식을 하면 체력이 회복됩니다.
- 체력이 모두 소진되거나 하루 턴을 모두 사용하면 다음 날로 넘어갑니다.
- 총 30일이 지나면 게임이 종료되고 최종 결과가 출력됩니다.

## 엔딩 조건

각 과목 능력치가 일정 기준 이상일 경우 대학 합격 엔딩이 출력됩니다.

- 모든 과목 480 이상: 서울대 합격
- 모든 과목 470 이상: 연세대 합격
- 모든 과목 460 이상: 고려대 합격
- 모든 과목 450 이상: 서강대 합격
- 모든 과목 440 이상: 성균관대 합격
- 모든 과목 430 이상: 한양대 합격
- 모든 과목 420 이상: 중앙대 합격
- 그 외: 불합격

## 사용 기술

- Java
- JavaFX

## 클래스 구성

- `MainMenu`
  - 게임 메인 메뉴 화면
  - 음악 재생 및 게임 시작 / 게임 방법 버튼 제공

- `Main`
  - 실제 게임 화면 구성
  - 상태창, 버튼, 다음 날 진행, 엔딩 출력 담당

- `Student`
  - 플레이어의 체력, 턴, 능력치 관리
  - 공부 / 휴식 로직 처리

- `GameManager`
  - 날짜 진행
  - 게임 종료 여부 판단
  - 최종 엔딩 판정

## 실행 환경

- Java
- JavaFX
- Eclipse

## 느낀 점

JavaFX를 활용해 화면 구성, 버튼 이벤트 처리, 상태 관리,  
게임 진행 로직을 직접 구현해보면서  
GUI 프로그램이 어떻게 동작하는지 배울 수 있었습니다.

또한 단순한 콘솔 프로그램이 아니라  
이미지, 음악, 버튼, 상태창을 함께 활용해  
하나의 작은 게임 형태로 완성해봤다는 점에서 의미가 있었습니다.
